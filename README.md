# Web-Speech-API를 활용한 프로젝트
### 기능 1. 국가별 발음 비교하기
단어나 문장을 입력하고 국가를 선택하면 국가별 발음을 들을 수 있습니다.
### 기능 2. 내 영어 발음 확인하기
Start STT 버튼을 누르고 단어나 문장을 말하면 들리는대로 문장이 입력되고 원어민 발음이 나옵니다.
Stop 버튼을 통해 멈출 수 있습니다.
